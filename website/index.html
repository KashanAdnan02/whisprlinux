WHISPRLINUX(1)
whisperlinux — offline push-to-talk dictation for Linux

Description

Whispr Linux listens for a global hotkey chord, records your voice locally, transcribes with faster-whisper, and injects text at the focused cursor — on both X11 and Wayland.

Features

  • 100% offline — faster-whisper runs on your CPU or NVIDIA GPU. No subscription, no telemetry.
  • Push-to-talk — hold Alt + Super + Shift, speak, release. Text appears instantly.
  • evdev hotkeys — reads keys at the kernel level, works across compositors and DEs.
  • xdotool / ydotool — text injection for X11 and Wayland respectively.
  • System tray — idle, recording, and transcribing states in your panel.
  • Autostart — optional systemd user service for login startup.

Configuration

Config file: ~/.config/whisperlinux/config.json

  • hotkey_modifiers — default Alt+Super+Shift
  • model_size — tiny.en, base.en, small.en, medium.en, large-v3
  • device — cpu or cuda
  • language — language code (default en)
