Quick install

Ubuntu / Debian

curl -fsSL https://whisperlinux.org/install.sh | bash

Or download the .deb and run:

sudo apt install ./whisperlinux_1.0.0_amd64.deb
whisperlinux --check

Any Linux (tarball)

tar -xzf whisperlinux-1.0.0-linux-amd64.tar.gz
cd whisperlinux-1.0.0-linux-amd64
./install.sh
whisperlinux --check

How to use

  1. Log out and back in after install (required for keyboard permissions).
  2. Run whisperlinux or enable autostart: systemctl --user enable --now whisperlinux.service
  3. Hold Alt + Super + Shift, speak, then release.
  4. Transcribed text is typed wherever your cursor is focused.

Features

100% offlineLocal Whisper AI — no API keys, no cloud.
X11 & WaylandKernel-level hotkeys work everywhere.
Push-to-talkHold hotkey to record, release to type.
System trayVisual status: idle, recording, transcribing.

Requirements

  • Linux x86_64, Python 3.10+ (included in tarball / .deb setup)
  • Microphone
  • xdotool (X11) or ydotool + ydotoold (Wayland)
  • User must be in the input group (installer handles this)
